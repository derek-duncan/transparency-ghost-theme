A) Posting - top

While creating a post, there are a few things you may implement to customize the post's content.

If you would like to edit the color or background image of the cover image, start your post with the snippet below.

;;youColorHexOrBackgroundImageUrl;;

Here is a more concrete example

          ;;ACBC65;; //Color Hex or this could work: ;;url('../images/image_name.png');;

          Lorem ipsum dolor sit amet, consectetur adipisicing elit.

This syntax with ensure you can replace the color/image on each post. This will automatically be removed from the post as well.

To add a description of the image, write it where the alt should be and it will be placed below the image

          ![ALT HERE](URL HERE)

B) CSS Files and Structure - top

I'm using four CSS files in this theme. "normalize.css" is a generic reset file. Many browser interpret the default behavior of html elements differently. By using a general reset CSS file, we can work round this. This file also contains some general styling, such as anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

"screen.css" contains all of the specific stylings that should be applied to every page on the website.

"index.css" contains all the styling for the index view "/".

"post.css" contains all the styling for the each post. "/".

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Beware when editing the css that it could destroy the styling of the blog.
